主管權限檢查

請先在「員工登入」頁面用你的手機號碼登入，再回到這裡按下方按鈕檢查你是否擁有主管權限。

公務機模式

開啟後，這台裝置（不管是員工登入頁還是主管後台）都會**只用本機儲存**來運作：員工送出資料不會強迫跳LINE，主管在「本機待審清單」直接統一審核。適合固定放在店裡、大家輪流用的公務手機/iPad。你自己的手機建議保持關閉。

目前是檢視模式（唯讀）

這台裝置沒有設定 GitHub Token，可以查看所有資料，但無法核准申請、開通員工或做任何寫入操作。若要正式寫入，請在有設定 Token 的那台裝置（例如公司公務機）上操作。

本機待審清單

列出這台裝置上，所有人累積但尚未經主管處理的申請，適合公用機情境：多人輪流用同一台裝置排休，最後由主管在這台機器上統一審核上線。
